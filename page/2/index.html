PAT自动签到 2019-11-25|code|实践

需求

去pat商店自动签到，攒50元代金券。

实现

通过使用python selenium + win计划任务

需要提前做的准备

  1. 安装python
  2. pip3 install selenium
  3. 安装ChromeDriver
  4. 把chromedriver.exe放到环境变量中，能够直接执行,我是直接放到scoop中的shims中,这里可以直接加入环境中

我先尝试模拟登录，发现验证码阶段无法自动通过，保存相关的cookie也没办法，最后的解决方案是使用原本的浏览器中的数据，来直接跳过登录，因为测试浏览器是新开的，没有任何配置，而让测试浏览器使用原本的数据会产生一些同步问题，所以我拷贝了一份浏览器数据专门给pat签到使用

关于如何通过xpath找相关的元素，chrome上有相关插件

另外，如何实现电脑自动化启动PAT签到.py文件

想法

关于web的cookie和session相关的知识还要注意去学习
